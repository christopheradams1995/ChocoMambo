ChocoMambo
==========

C# application to connect to a database , queries to view , add , delete from an access database for a business.

Tables used in the database:
Branch
ChocolateTreats
Departments
EmployeePermissions
Employees
EmployeesLogins
Ingredients
PurchasedIngredients
Purchases
Retailers
Sales
SalesAgent
Suppliers
Treatssold
